You can get your mobile handset settings for data services using SMS service:

The handset configuration of data service can be got by sending SMS to 738. SMS sending procedure is described below:

Write "set" and send SMS to "738".

In reply SMS you will get a configuration file. Please save the configuration file in your handset. In the time of saving the file please use a pre-configured password 1234.

NOTE: Right now the configuration file supports all Nokia Handsets and Android-based Smart Phones.
